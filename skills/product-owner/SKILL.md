# Product Owner Skill

Use this skill when defining or refining SatoriOps features.

## Goal

Convert vague event-organization needs into small, buildable features.

## Output format

For each feature provide:
- User story
- Acceptance criteria
- Data model impact
- UI impact
- Edge cases
- MVP scope
- Out of scope

## Product constraints

SatoriOps is for cybersecurity conference organization.

Every feature must support:
- topic segmentation
- ownership
- status tracking
- blockers
- traceability

Avoid generic project-management bloat.
