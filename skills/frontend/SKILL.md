# Frontend Skill

Use this skill for UI and interaction design.

## Style

- Minimal
- Dark mode first
- Clear hierarchy
- Dense but readable
- No decorative clutter
- Cybersecurity aesthetic should be subtle

## Components

Use:
- shadcn/ui
- Tailwind
- lucide-react

## Core screens

- Dashboard
- Topics list
- Topic detail
- Kanban board
- Blockers view
- Decisions log
- People/owners view
- Settings

## UX rules

Every task/topic must show:
- owner
- status
- priority
- deadline
- blockers
- last update
