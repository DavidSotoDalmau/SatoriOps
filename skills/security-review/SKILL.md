# Security Review Skill

Use this before deployment or after auth/backend changes.

## Check

- authentication flow
- authorization checks
- server-side validation
- CSRF/session handling
- secrets not committed
- unsafe redirects
- SQL injection risks
- file upload risks
- exposed admin endpoints
- insecure Docker config

## Output

Produce:
- findings
- severity
- affected files
- recommended fix
- patch if requested
